# QR Code Badge
Web application used to generate QR codes and place in pritable PDFs with configurable layout. 

A QR code can contain a link or a email message with to: cc: title and message body. Inputs for the qr code generation is provided by a used in a table view with predefined columns such as QR code type, link or email message parameters. User can import data from a CSV file as well. 

PDF templates are configurable and allow placement of the QR codes, title and sub-title (also configurable in the table).

Application is using existing stable frameworks where possible.
