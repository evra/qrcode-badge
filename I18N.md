# Internationalization (i18n)

This application now supports multiple languages through i18next and react-i18next.

## Supported Languages

- **English (en)** - Default language
- **German (de)** - Deutsch

## Features

- Language switcher in the app bar (top right corner)
- All UI text is translated including:
  - Navigation tabs
  - Button labels
  - Form fields
  - Notifications and error messages
  - Help text and instructions
  - Table headers and content
  
- Language preference is persisted in browser's localStorage
- Automatic language detection on first visit (defaults to browser language if supported, otherwise English)

## How to Use

1. Click the language selector dropdown in the top right corner of the app bar
2. Select your preferred language (English or Deutsch)
3. The entire interface will immediately update to show text in the selected language
4. Your language preference will be saved and remembered on your next visit

## For Developers

### Adding a New Language

1. Create a new JSON file in `src/i18n/locales/` (e.g., `fr.json` for French)
2. Copy the structure from `en.json` and translate all values
3. Import the new language file in `src/i18n/config.ts`
4. Add it to the resources object in the i18n configuration
5. Add a menu item for the new language in `App.tsx` language selector

### Translation Structure

All translations are organized hierarchically in JSON files:

```json
{
  "app": {
    "title": "Application Title",
    ...
  },
  "tabs": { ... },
  "notifications": { ... },
  ...
}
```

### Using Translations in Components

```typescript
import { useTranslation } from 'react-i18next';

function MyComponent() {
  const { t } = useTranslation();
  
  return <div>{t('app.title')}</div>;
}
```

### Translations with Variables

```typescript
// In component
t('notifications.importSuccess', { count: 5 })

// In translation file
"importSuccess": "Successfully imported {{count}} entries"
```

## File Structure

```
src/
  i18n/
    config.ts           # i18next configuration
    locales/
      en.json          # English translations
      de.json          # German translations
```
